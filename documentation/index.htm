HTML Template

About

This is a documentation for Housing HTML template. It means that it is a static HTML template and it's not compatible with WordPress. You need to have some basic knowledge about HTML, CSS and JavaScript language.

Use an API Key!

In order to run this template with Google Maps properly, you need to obtain a Google Maps API Key

Background Class

You can set a background image or color for .block or any other relative positioned element.

Background Color

Use data-background-color attribute for color. 

<section class="block">
    ...
    <div class="background" data-background-color="#fff"></div>
</section>

Background Image

<section class="block">
    ...
    <div class="background">
        <div class="background-image">
            <img src="assets/img/background-image.jpg" alt="">
        </div>
    </div>
</section>

Important: If your background image is dark, use has-dark-background class

Google Maps API Key

A first step if you want to use Google Maps is to get an API key from this link.. Otherwise you'll get grey map and Oops! Something went wrong. message.

Then on every page where with the map find this code at the bottom of the page:

<script type="text/javascript" src="http://maps.google.com/maps/api/js?key=YOUR_API_KEY&libraries=places"></script>

Where replace text YOUR_API_KEY with your own API you've get.

Google Map

To display Google map you need to create an element with the map class and with unique ID and .height-***px class for defining a height of the map. Read more here about height CSS class. 

<div class="map height-300px" id="your-map-id"></div>

Then at the bottom of the page after <script> list enter this code: 

<script>
    var latitude = 51.511971;
    var longitude = -0.137597;
    var markerImage = "assets/img/map-marker.png";
    var mapTheme = "light";
    var mapElement = "your-map-id";
    simpleMap(latitude, longitude, markerImage, mapTheme, mapElement);
</script>

Note Map can have light and dark theme. If you use dark theme, use also markerImage = "assets/img/map-marker-w.png";

Parallax Background

To use parallax effect on your background image it is recommended to use image with at least 2000px width and with high that will fill desired section. Than use data-parallax-"scroll" attribute and data-parallax-speed="1-5". Then child element has to have parallax-element class.

Then on every page where with the map find this code at the bottom of the page:

<div class="background-wrapper" data-parallax="scroll" data-parallax-speed="3" data-background-color="#f1f1f1"">
    <div class="background--image opacity-10 parallax-element">
        <img src="assets/img/slide-02.jpg" alt="">
    </div>
</div>

Owl Carousel

You can create a carousel without editing a JavaScript. All you need to is to set data- parameters. These data-attributes you can set:

data-owl-items

number of displayed items. Default 1

data-owl-nav

display navigation arrows. Default 0

data-owl-dots

display navigation dots. Default 0

data-owl-center

center the slide. Default 0

data-owl-loop

loop the carousel. Default 0

data-owl-margin

margin between slides. Default 0. (If doesn't work, set margin via CSS)

data-owl-auto-width

dynamic width based on slide width. Default 0

data-owl-nav-container

container for navigation. Default 0

data-owl-autoplay

auto play slides. Default 0

data-owl-fadeout

fade animation between slides. Default 0
